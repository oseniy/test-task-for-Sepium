## Стек

| Технология | Применение |
|---|---|
| **HTML5** | Семантическая разметка, SVG-спрайт иконок |
| **CSS3** | CSS-переменные (дизайн-токены), Flexbox, Grid, анимации, `@media` для адаптива |
| **JavaScript** | Модульный паттерн (IIFE), логика карточек, лайки, переключение модификаций |
| **jQuery 3.7** | DOM-манипуляции и обработка событий |
| **Swiper 11** | Карусель изображений товаров |
| **Fancybox 5** | Лайтбокс для просмотра фото |
| **Live Server** | Локальный dev-сервер |

## Архитектура

```
css/
  variables.css      ← дизайн-токены (цвета, типографика, отступы)
  reset.css          ← сброс стилей
  base.css           ← базовые стили
  product-listing.css← сетка каталога
  product-card.css   ← компонент карточки
  animations.css     ← анимации и переходы
js/
  app.js             ← точка входа, инициализация модулей
  slider.js          ← модуль Swiper-слайдера
  card.js            ← модуль карточки (лайк, модификации, клик)
  fancybox-init.js   ← модуль лайтбокса
```

CSS разбит по компонентам, все дизайн-токены вынесены в `variables.css`. JS организован через IIFE-модули с публичным API — каждый файл отвечает за одну функциональность.

## Запуск

```bash
npm install
npm start
# http://localhost:3000
```
